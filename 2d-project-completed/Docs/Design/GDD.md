# GDD — 2D Survival (작업용)

> **Game Identity:** 로비에서 모드를 고른 뒤, 자동 공격과 런 한정 빌드로 몹의 압박을 버티고 웨이브 또는 시간 목표를 달성하는 2D 뱀파이어 서바이버류 액션 게임. 아레나 모드는 웨이브 변이와 제한된 인벤토리 판단을 통해 더 강한 전장 압박을 제공한다.
>
> **상태:** 초안 · **갱신:** 2026-05-28  
> **범위:** 공개 데모(MVP) 기준. 구현 상세·수치는 Wiki/Architecture로 위임.

**핵심 결정:** Survivors-like 데모 정체성을 유지하고, **아레나 모드에 한해** Simulacrum(Risk of Rain 2)풍 **웨이브 변이**와 **인벤토리 판단 강화**를 부분 도입한다. 상시 시스템이 아니라 특정 웨이브 modifier로만 적용한다.

## Right — Play & Production

### Main Aim

플레이어가 한 판 안에서 **빌드를 키우며 압박을 이겨 목표 지점까지 도달**하는 것.

| 모드 | Main Aim |
|------|----------|
| **아레나 (데모 기본)** | 1~10웨이브를 텔레포터로 시작·클리어하며 최종 웨이브(10) 승리. 일부 웨이브에서는 변이 규칙과 인벤토리 판단까지 감당 |
| **서바이벌 (비교 모드)** | 시간이 갈수록 강해지는 몹 속에서 **30분** 생존 후 스테이지 클리어 |

### Main Mechanic

**이동(WASD) + 조준/자동 공격(마우스·G 토글) + 회피(대시)** 으로 몹을 처치하고, **무기·장비 선택 + 인벤토리 정리**로 딜과 생존을 올린다.

- **아레나:** 웨이브 클리어 시 **무기 3택1** + **골드 상자** + **웨이브 변이 규칙**으로 런 빌드 보강
- **아레나 변이:** 상시 시스템이 아니라 **특정 웨이브 modifier**로만 적용 (아래 [아레나 웨이브 Modifier](#아레나-웨이브-modifier-mvp-lite))
- **서바이벌:** 처치 → XP/골드 → **레벨업 3택1** 무기 (기존 구조 유지)
- **인벤토리:** 장비 배치·버리기·전투 세트 스왑 구조 유지. 아레나에서는 웨이브 보상·장비 선택·가방 압박으로 **판단 빈도**를 높임

### 아레나 웨이브 Modifier (MVP-lite)

**적용 범위:** 아레나만. 서바이벌 타임라인에는 넣지 않는다.  
**역할:** 난이도 스파이크·공간 판단·빌드 정리를 한 웨이브 안에 압축. **별도 승리 조건 아님.**

| Modifier (후보) | 플레이어 체감 | GDD 제약 |
|-----------------|---------------|----------|
| **국소 위험 지대** | 맵 일부에 지속 피해·디버프 구역 | 전장 전체를 위험으로 만들지 않음. 이동·대시로 회피 가능해야 함 |
| **제한 안전 구역** | 작은 안전 구역만 회복·대기 가능 | 상시 안전지대 생존 게임이 되면 안 됨. 웨이브 종료 시 해제 |
| **회복 감소** | 체력 픽업·자연 회복 효율 하락 | 인벤/상자로 보완 가능한 수준. 즉사 트랩 금지 |
| **엘리트 강화** | 해당 웨이브 엘리트 스폰·스탯 상승 | 기존 elite/special/boss와 역할 중복 최소화 |

**데모 MVP-lite:** 위 4종 중 **1~2종만** 1~10웨이브에 배치해 검증. 나머지·복합 modifier는 Later.

**배치 확정(2주차):** 웨이브 **6** = 국소 위험 지대, **9** = 회복 감소 — [`GameRules.md`](../Wiki/GameRules.md). 구현 3주차.  
**UI:** 웨이브 시작 전 modifier 이름·짧은 설명·아이콘. 위험/안전 구역은 색·테두리로 시인성 확보.  
**구현·수치:** Wiki `GameRules` / `Mobs` / Architecture `GameLoop` — GDD는 의도만 고정.

### Core Loop

**아레나 (데모 기본)**

```text
로비 → 아레나 선택 → 시작 무기 3택1 → (일시정지 해제)
  → 중앙 텔레포터 E → (modifier 안내) → 웨이브 스폰
  → 처치 + 전장 규칙 대응(위험 구역 / 안전 구역 / 엘리트 변이)
  → (스폰 완료 & 필드 0) 웨이브 클리어
  → 무기 3택1 → 골드 상자 구매 → 인벤토리 정리 → 텔레포터 재활성
  → 반복 … → 10웨이브 클리어 → 스테이지 클리어
  ‖ 체력 0 → 게임오버
```

**서바이벌 (비교)**

```text
로비 → 서바이벌 선택 → 시작 무기 3택1 → 스폰 시작
  → 처치 → XP/골드 → 레벨업 3택1 → 시간축 밸런스 강화
  → 30분 도달 → 클리어  ‖  체력 0 → 게임오버
```

### Win / Lose

| 모드 | Win | Lose |
|------|-----|------|
| **아레나** | MVP **10웨이브** 클리어. 변이 웨이브는 난도 강화 요소이지 별도 승리 조건 아님. 이후 30/50/70/100 확장은 후속 | 플레이어 HP ≤ 0 → 게임오버(씬 리로드) |
