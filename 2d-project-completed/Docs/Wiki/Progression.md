## 성장 구조

서바이벌 성장은 “몹 처치 → 경험치 수집 → 레벨업 → 무기 획득” 흐름이 중심이다. 아레나는 처치 경험치 대신 웨이브 클리어 시 무기 획득과 골드 상자 구매를 제공한다.

1. 서바이벌에서 몹을 처치하면 경험치 오브와 골드가 떨어진다.
2. 플레이어가 오브를 수집하면 경험치를 얻는다.
3. 경험치가 목표치를 넘으면 레벨업한다.
4. 무기 획득 UI에서 무기 후보를 고른다.
5. 선택한 무기는 런 인벤토리에 들어가며, 빈 weapon 슬롯이 있으면 자동 배치된다.
6. 활성 전투 세트의 weapon 슬롯에 장착된 무기만 실제 공격에 사용된다.

## 경험치

| 항목 | 규칙 |
|------|------|
| 드랍 | 몹 종류와 현재 진행 구간에 따라 경험치량이 정해진다. |
| 수집 | 경험치 오브가 플레이어의 수집 범위에 들어오면 자석처럼 빨려 온다. |
| 레벨업 | 누적 경험치가 기준치를 넘으면 레벨업하고 무기 획득 UI가 열린다. |
| 아레나 | 몹 처치 경험치는 0이며, 경험치 오브 대신 웨이브 종료 시 무기 획득과 상자 구매 보상 단계가 열린다. |
| 클리어 사망 | 30분 클리어로 사라지는 몹은 XP를 드랍하지 않는다. |

## 골드

| 항목 | 규칙 |
|------|------|
| 드랍 | 몹 처치 보상 XP를 기준으로 골드가 계산된다. |
| 수집 | 골드 코인은 경험치 오브처럼 자석 수집된다. |
| 표시 | HUD에 현재 골드가 표시된다. |
| 소비 | 웨이브 클리어 보상 단계에서 런 한정 상자를 여는 비용으로 사용한다. |
| 초기화 | 골드는 런 중 자원이며 클리어, 패배, 로비 복귀, 새 런 시작 시 저장하지 않는다. |

## 상자

상자는 골드를 소비해 현재 런 인벤토리에 장비를 추가하는 보상이다. 상자에서 얻은 장비는 현재 런 안에서만 장착, 교체, 보관할 수 있으며 영구 세이브에는 쓰지 않는다. 대상 장착 슬롯이 비어 있으면 획득 즉시 자동 장착되고, 이미 차 있거나 offhand가 양손 무기로 막혀 있으면 가방에 들어간다. 장비 효과는 실제 장착 슬롯에 들어간 뒤부터 적용된다.

| 항목 | 규칙 |
|------|------|
| 비용 | 상자를 열 때 골드를 차감한다. 골드가 부족하면 열 수 없다. |
| 결과 | 상자 1개는 아이템 1개를 지급한다. |
| 전체 상자 | 모든 슬롯 아이템이 나올 수 있다. |
| 부위 상자 | 상자 이름의 슬롯만 나온다. 예: `offhand` 상자는 `offhand`만 지급한다. |
| 등급 | 아레나 MVP 1~10웨이브에서는 `Common`과 `Uncommon`만 나온다. |
| 중복 | 같은 런 안에서 동일 `item_id`는 중복 지급하지 않는다. |

## 픽업

| 픽업 | 현재 규칙 |
|------|-----------|
| 경험치 오브 | 주요 성장 자원. 풀링되어 재사용된다. |
| 골드 코인 | 런 중 수집 자원. 풀링되어 재사용된다. |
| 자석 | 낮은 확률로 드랍. 필드의 경험치와 골드를 끌어당긴다. |
| 체력 | 낮은 확률로 드랍. 플레이어 체력을 회복한다. |

## 한 판 안 성장의 목표

- 초반에는 무기 종류가 늘어나는 재미를 준다.
- 중반에는 선택한 빌드의 강점과 약점이 보인다.
- 후반에는 보스와 밀도 이벤트를 버틸 만큼 성장했는지 확인한다.
- 데모 기준으로는 “선택이 빌드를 바꿨다”는 체감이 가장 중요하다.

## 현재 큰 갭

| 갭 | 영향 |
|----|------|
| 보유 무기 강화 부족 | 레벨업 보상이 신규 무기 획득에 치우친다. |
| 메타 진행 없음 | 여러 판 반복 동기가 약할 수 있다. |
| 패시브·액세서리 성장 없음 | 빌드 다양성이 무기 종류에만 의존한다. |

## 데모 기준

데모에서는 골드를 숨기지 않고 런 한정 상자 비용으로 사용한다. 상자는 메타 진행이 아니라 한 판 안의 장비 빌드 선택지를 늘리는 역할에 집중한다.

## 미정 / 후속

- 레벨업 곡선과 필요 경험치 튜닝.
- 엘리트/보스 전용 경험치와 드랍 비주얼.
- 런 간 영구 해금.
- 런 종료 리포트 JSON 저장.
- 플레이타임, 처치 수, 무기 획득 통계 수집.
