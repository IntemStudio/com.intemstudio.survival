## 공격 전달 방식

무기는 `WeaponData`의 타입과 movement에 따라 탄환, 근접 관통 탄, 마법 탄, 투척체, 영역 존, 궤도 공격으로 전달된다. 이 문서는 플레이어가 체감하는 이동과 판정 규칙을 다루고, 구현 세부사항은 아키텍처 문서에서 관리한다.

| 방식 | 규칙 |
|------|------|
| 탄환 | 사거리 안 최근접 몹을 향해 발사한다. |
| 근접 관통 탄 | 판정은 짧은 발사체로 유지한다. 비주얼은 총알이 아니라 검기/베기 궤적처럼 보여 근접 공격 모션에 가깝게 느껴져야 한다. |
| 마법 탄 | 직선, 유도, 궤도 등 마법 계열 movement를 사용한다. |
| 투척체 | 목표 방향 또는 착지 지점을 향해 날아가며, 일부는 돌아오거나 영역 존을 만든다. |
| 영역 존 | 착지 후 원형 또는 사각 영역을 만들고, 영역 안 몹에게 피해를 준다. |
| 궤도 | 플레이어 주변을 돌며 충돌한 몹에게 피해를 준다. |

## Movement 타입

| 타입 | 사용처 | 플레이 규칙 |
|------|--------|-------------|
| `Straight` | 일반 탄환, 일부 투척체 | 조준 방향으로 직선 이동한다. |
| `StraightPierce` | 기본 근접 관통 탄 | 조준 방향으로 직선 이동하며 여러 몹을 관통할 수 있다. |
| `Return` | 창, 사이드 소드 등 | 조준 방향으로 직선 이동한 뒤 플레이어에게 직선으로 돌아온다. |
| `CurvedReturn` | 작물 낫, 바스타드 소드 등 | 조준 방향을 기준으로 긴 타원형 궤적을 그리며 나갔다가 반대쪽 곡선으로 돌아온다. |
| `Decelerate` | 낫, 녹슨 부채 등 | 조준 방향으로 나란히 날아가며 속도가 자연스럽게 줄고, 거의 멈춘 뒤 짧게 남았다가 사라진다. |
| `Homing` | 유도 마법 탄 | 이동 중 가까운 몹을 향해 방향을 보정한다. |
| `Arc` | 연금 투척체 등 | 착지 지점까지 포물선처럼 이동한 뒤 효과를 만든다. |
| `Orbit` | 왕의 성경, 가시 철퇴 등 | 플레이어 주변을 회전하며 overlap 판정으로 피해를 준다. |

## 근접 발사체

근접 무기는 이름과 역할은 melee지만, 대부분 `MeleeProjectile` 발사체 모델을 사용한다. 근접 스윙감은 검기형 비주얼과 movement 차이로 표현한다. 예외적으로 가시 철퇴처럼 궤도 역할인 근접 무기는 `Orbit` companion을 사용한다.

| 무기 | 현재 movement |
|------|---------------|
| Spiky Flail | `Orbit` |
| Sickles | `Decelerate` |
| Rusty Fans | `Decelerate` |
| Spear | `Return` |
| Side Sword | `Return` |
| Crop Scythe | `CurvedReturn` |
| Bastard Sword | `CurvedReturn` |

기본 근접 무기는 `StraightPierce`를 사용한다. `Return`과 `CurvedReturn`은 왕복 시 다시 같은 몹을 칠 수 있으므로, 체감 피해와 관통 수 튜닝을 함께 확인한다. `Decelerate` 계열은 2발을 중심선 좌우 같은 거리에서 나란히 생성하고, 속도 조정으로 체류 시간과 도달 거리를 맞춘다.

## 충돌과 피해 귀속

- 플레이어 발사체는 벽, 나무 등 환경 장애물에 막힌다.
- 일반/관통/투척/부메랑 계열은 장애물 충돌 또는 사거리 종료 시 반환된다.
- 폭발형 탄, 폭발 마법, 연금 투척체는 충돌 지점에서 폭발 또는 영역 존을 만든다.
- 독/상태 피해는 원본 무기 귀속을 유지해 피해 통계에 반영한다.

## 품질 체크

- F6 테스트 아레나에서 각 movement 대표 무기를 Equip하고 더미/기본 몹에게 피해가 들어가는지 확인한다.
- 벽과 나무에 닿았을 때 반환, 폭발, 영역 생성 규칙이 기대대로 동작해야 한다.
- `Return`과 `CurvedReturn`은 왕복 경로에서 재타격 체감이 과하지 않은지 확인한다.
- `Decelerate`는 멈춘 뒤 잔류 시간이 길거나 짧지 않은지, 병렬 간격이 몹 크기와 맞는지 확인한다.
- 자동 공격 on/off·사거리 밖(몹 없음)에서 발사체 생성이 멈추고, 궤도 피해 판정은 토글을 따라야 한다.
