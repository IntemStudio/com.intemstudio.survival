## Arena — 웨이브 Modifier (MVP-lite)

GDD·Wiki에 기획만 반영됨. 코드 없음. 데모에서는 **4종 후보 중 1~2종**, **1~10웨이브 중 1~2개 웨이브**만 검증.

- [ ] **modifier 데이터·배치표** — **기획:** 웨이브 6=국소 위험 지대, 9=회복 감소 (`GameRules.md`). 코드·리소스 단일 출처 정리.
- [ ] **텔레포터 시작 UI** — 웨이브 시작 직전 이름·한 줄 설명·아이콘. Esc 일시정지와 충돌 없게.
- [ ] **국소 위험 지대 (후보 1)** — 맵 일부 지속 피해/디버프. 전장 전체 위험 금지, 이동·대시 회피 가능.
- [ ] **제한 안전 구역 (후보 2)** — 소형 회복·대기 구역. 웨이브 종료 시 해제. 상시 안전지대 생존 방지.
- [ ] **회복 감소 (후보 3)** — 체력 픽업·자연 회복 배율. 즉사 트랩 금지.
- [ ] **엘리트 강화 (후보 4)** — 해당 웨이브 엘리트 스폰·스탯. 기존 elite/special/boss와 역할 겹침 검토.
- [ ] **위험/안전 구역 시인성** — 색·테두리·짧은 경고 SFX. F5 아레나에서 튜토리얼 없이 읽히는지 수동 QA.
- [ ] **modifier 라이프사이클** — 웨이브 클리어·패배·텔레포터 대기 시 해제. 다음 웨이브에만 재적용.
- [ ] **F5 검증** — modifier 웨이브 1종 이상 포함 빌드로 1~10웨이브 또는 해당 구간 무중단 플레이.

**후속 (데모 밖):** modifier 복합, 11웨이브+ 확장, Simulacrum식 독립 모드, 서바이벌 타임라인 적용.

- [ ] **엘리트** — 전용 기믹, 등장 연출, 보상 차별화.
- [ ] **위협도 지표 활용** — 현재 밸런스 데이터의 threat 값을 피해, 스폰, HUD 중 어디에 연결할지 결정.
- [ ] **몹 종류 값 활용 확대** — AI, 보상, UI 분기에 적극 사용할지 결정.
- [ ] **레이피어 “En Garde”** — 웨이브 시작 시 8초 동안 공격 속도 +20% 버프를 적용하는 버프 시스템 구현.

## Ideas

- 보스 처치 시 일시 무적, 전체 경험치, 상자 드롭.

- 패시브 슬롯 교체·리셋(런 중 1회) — 빌드 실험용.
- 5분마다 30초 고밀도 같은 시간 제한 이벤트.
- 소나무 파괴, 불 장판 등 환경 상호작용.
- 플레이어 넉백.
- [ ] F6 스태미나·대시·회복 튜닝 UI — [`Architecture_Player.md`](Docs/Architecture/Architecture_Player.md) 후속.
