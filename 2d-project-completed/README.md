# Godot 2D Survival

Godot 4.5.2 기반 2D 뱀파이어 서바이버류 프로젝트입니다.

## 빠른 시작

| 동작 | 씬 |
|------|-----|
| F5 (메인) | `game_lobby.tscn` → 서바이벌/아레나 런 |
| F6 (테스트) | `test_arena.tscn`을 **현재 씬**으로 연 뒤 실행 |
